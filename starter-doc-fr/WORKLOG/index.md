# WORKLOG

## Concepts

* [Current task](./current-task.md) - Point de reprise partagé entre assistants IA pour suivre la tâche courante, son statut, les fichiers touchés, les vérifications et la prochaine action.
* [ROADMAP](./ROADMAP.md) - Exemple de roadmap de tickets , backlog ordonné pour livrer progressivement le produit, avec un critère d'acceptation visible par ticket. Cette roadmap est écrite par l'utilisateur, elle est fournie à titre d'exemple.
